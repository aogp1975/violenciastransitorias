Uso de Geolocalización

Este sitio utiliza la ubicación de tu dispositivo con fines artísticos y de investigación,
no se recolectan datos personales, ni se comparte tu ubicación a terceros,
únicamente funciona para centrar el mapa y para registrar
la ubicación de los stickers colocados.
Al permitirlo, aceptas el uso de tus datos de geolocalización con estos fines.
Si prefieres no compartirla, aún puedes navegar por el sitio libremente.

Acerca del proyecto

Violencias transitorias es un proyecto de intervención y apropiación del espacio público con el acoso callejero como motivo de acción. Buscamos recuperar las calles y marcarlas a través de stickers, alzar nuestra voz y nombrar las agresiones. Este mapa es un espacio para denunciar, actuar y expandir la colaboración. La calle también es nuestra, tenemos derecho de habitarla.
